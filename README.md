# Low-Speed Open-Circuit Subsonic Wind Tunnel
![Status](https://img.shields.io/badge/status-in%20development-yellow)
![License](https://img.shields.io/badge/license-MIT-blue)
![Platform](https://img.shields.io/badge/platform-Raspberry%20Pi%20Pico%20W-red)
![CFD](https://img.shields.io/badge/CFD-OpenFOAM-green)

A self-designed, fabricated, and instrumented aerodynamic 
test facility for comparative force measurement on 1:64 scale 
vehicle and missile body models at Reynolds numbers of 
20,000–50,000.
